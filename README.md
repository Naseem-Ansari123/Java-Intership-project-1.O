# 🚀 Java Internship Project 1.0

A simple and beginner-friendly Java REST API project built using Spring Boot.  
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations for:

- 👨‍💼 Employee
- 🎓 Student
- 🛍️ Customer

---

# ✨ Features

✅ Create new records  
✅ Get all records  
✅ Update existing records  
✅ Delete records  
✅ RESTful API design  
✅ Maven project structure  
✅ JSON request & response  
✅ Beginner-friendly codebase  

---

# 🛠️ Technologies Used

- ☕ Java
- 🌱 Spring Boot
- 📦 Maven
- 🔗 REST API
- 🧪 Postman
- 🗄️ JSON Data
